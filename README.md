# JSONDerulo

A JSON format that is for serial and forreal.


## Usage

```JSON

$ derulo [--version, -v]
  "JSONDerulo version JSONDerulo" 
  
```

# Basic Usage
```JSON
derulo --say --output=json
  {
    "JSONDerulo": "whatcha say"
  }

# Syntactic Sugar
derulo --say --output=json
  {
    "JSONDerulo": "whatcha say"
  }
}
```

```JSON
# Streamig JSON FORMAT
derulo --stream solo
{
  "JSONDerulo": 
{
    "derulo":
      [
```
