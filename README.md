# JSONDerulo

A JSON format that is for serial and forreal.


## Usage

derulo --version
  "what'cha say"

derulo --say
{
  "derulo":
    ["IS THERE SUCH A THING AS A CONNECTED CASTLE?", "“Women Lie, Men Lie, Numbers Don’t Lie!"]
}

derulo --stream
{
  [
    "Riding solo, soooooloooo", "yeah it's like",
        "S", "O", "L", "O",
        "S", "O", "L", "O",
        "S", "O", "L", "O",
        "S", "O", "L", "O",
  ]
}
